本文记录我个人目前在使用的一些“工具”，以及一些偏好。

- Linux发行版：CentOS 7.0
- 科学上网： `huhamhire-hosts`_ > `USTC LUG OpenVPN`_
- 版本控制: Git
- Git托管服务： `GitHub`_ > `USTC GitLab`_ > `GitCafe`_

  - USTC GitLab用于托管私有项目；

- Shell：zsh > bash
- 高级编程语言： C > Fortran

- Python IDE：Vim > `PyCharm Communitu Edition`_
- 编程字体： `Droid Sans Mono`_
